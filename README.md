# dotmgr — Dotfiles Manager

CLI-инструмент для синхронизации конфигов и настройки рабочего окружения между Linux-машинами.

## Мотивация

Поддержка идентичного окружения на нескольких машинах (ноутбуки, виртуалки, VPS) требует повторяющихся ручных действий: копирование конфигов, установка одних и тех же пакетов, перенос скриптов.

dotmgr решает три задачи:
- **Reproducibility** — единый источник правды для настроек
- **Bootstrap** — быстрая раскатка окружения на новой системе
- **Propagation** — распространение изменений между машинами
